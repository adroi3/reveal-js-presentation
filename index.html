What is reveal.js?

  • reveal.js is a framework for easily creating a web-based presentation
  • There are different approaches to create them
  • You can use HTML
  • You can use Markdown

Pros/cons

Pros

  • Can be easily managed via a version control system (no binary code)
  • Can be widely customized with plugins and configurations
  • You only need a browser on the target machine
  • Changing the theme of the presentation don't produce trouble

Cons

  • Usage is more complex than PowerPoint
  • You can find more information about PowerPoint than about reveal.js

How can you use reveal.js?

  • Via the online editor
  • Set it up without a server
  • Set it up with a Node.js server
Setup without a server
-Simply download the latest release version
-Now you can edit the index.html site and display it via a browser
-You can't do everything with this option
